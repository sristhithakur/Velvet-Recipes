🍽️ Velvet Recipes – Food Recipe UI/UX Website
A modern frontend food recipe website focused on clean UI/UX design, smooth interactions, and responsive layout.
This project showcases a visually appealing recipe browsing experience built with HTML, CSS, and JavaScript.
The goal of the project is to demonstrate beautiful interface design, smooth user interaction, and responsive layouts for food and cooking platforms.


✨ Features
🍲 Elegant recipe card layout
🎥 Cinematic hero landing section
🖱️ Smooth hover animations
📱 Fully responsive design
🎨 Modern UI/UX styling
🔍 Recipe browsing experience
🧾 Clean typography with Google Fonts
⚡ Lightweight and fast loading
